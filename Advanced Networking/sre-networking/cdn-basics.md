# CDN (Content Delivery Network)

## Overview

CDN distributes content across multiple global servers.

---

## Why CDN Exists

To reduce:
- latency
- bandwidth usage
- server load

---

## How It Works

User → Nearest CDN Edge Server → Origin Server

---

## Example CDNs

- Cloudflare
- Akamai
- AWS CloudFront

---

## Cached Content

- images
- videos
- static files
- CSS/JS

---

## Real-world Example

User in India:
→ served from nearest edge server instead of US server

---

## Benefits

- faster load times
- reduced latency
- better scalability

---

## Troubleshooting

### Issue: stale content

Check:
- cache invalidation
- TTL settings
