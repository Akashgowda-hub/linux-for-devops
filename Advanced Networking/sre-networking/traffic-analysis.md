# Traffic Analysis

## Overview

Traffic analysis is the study of how data flows through a system.

---

## Why It Matters

Used for:
- performance optimization
- security monitoring
- incident detection

---

## Key Metrics

- request rate
- latency
- error rate
- bandwidth usage

---

## Tools

- tcpdump
- Wireshark
- Prometheus
- Grafana

---

## Real-world Example

Detecting:
- traffic spikes
- DDoS attacks
- slow endpoints

---

## Flow Analysis

Client → LB → Service → Database

---

## Troubleshooting

### Issue: traffic spike

- logs
