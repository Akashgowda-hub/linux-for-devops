# nftables

## Overview

Modern replacement for iptables.

---

## Why Important

Used in modern Linux distributions.

---

## List Rules

```bash
sudo nft list ruleset
```

---

## Add Rule

```bash
nft add rule inet filter input tcp dport 22 accept
```

---

## Advantages

- faster
- simpler
- unified framework

---

## Interview Questions

1. Difference between nftables and iptables?

---

## Summary

nftables is the modern Linux firewall system.
