# Disk Management

Disk management helps monitor storage devices and filesystems.

---

# lsblk

Show block devices.

```bash
lsblk
```

---

# fdisk -l

List partitions.

```bash
sudo fdisk -l
```

---

# df -h

Show filesystem disk usage.

```bash
df -h
```

---

# du -sh

Show directory size.

```bash
du -sh folder/
```

---

# mount

Mount filesystem.

```bash
mount /dev/sdb1 /mnt
```

---

# umount

Unmount filesystem.

```bash
umount /mnt
```

---

# blkid

Show UUID information.

```bash
blkid
```

---

# File Systems

Common Linux filesystems:

- ext4
- xfs
- btrfs

---

# Summary

Disk management is important for monitoring server storage usage.
