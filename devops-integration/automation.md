# Automation

Automation reduces manual work in DevOps.

---

# Common Automation Tools

- Shell Scripting
- Ansible
- Terraform

---

# Bash Automation Example

```bash
#!/bin/bash

systemctl restart nginx
```

---

# Ansible

Agentless automation tool.

Run playbook:

```bash
ansible-playbook playbook.yml
```

---

# Terraform

Infrastructure as Code tool.

```bash
terraform init
terraform apply
```

---

# Scheduled Automation

Use cron jobs:

```bash
crontab -e
```

---

# Benefits of Automation

- Faster deployments
- Reduced errors
- Better consistency

---

# Summary

Automation is a core part of DevOps workflows.
