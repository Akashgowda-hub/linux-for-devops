# CI/CD Basics

CI/CD automates software build, testing, and deployment.

---

# CI

Continuous Integration:

- Build code
- Run tests
- Detect issues early

---

# CD

Continuous Delivery/Deployment:

- Deploy applications automatically
- Reduce manual work

---

# Popular CI/CD Tools

- Jenkins
- GitHub Actions
- GitLab CI
- ArgoCD

---

# Basic CI/CD Flow

```text
Code → Build → Test → Deploy
```

---
