# File Commands

## ls

List files and directories.

```bash
ls
ls -la
```

---

## pwd

Print current working directory.

```bash
pwd
```

---

## cd

Change directory.

```bash
cd /home/user
```

---

## touch

Create empty file.

```bash
touch file.txt
```

---

## cat

Display file contents.

```bash
cat file.txt
```

---

## cp

Copy files/directories.

```bash
cp file1.txt file2.txt
```

---

## mv

Move or rename files.

```bash
mv old.txt new.txt
```

---

## rm

Delete file.

```bash
rm file.txt
```

---

## rm -r

Delete directory recursively.

```bash
rm -r folder/
```

---

## mkdir

Create directory.

```bash
mkdir project
```

---

## rmdir

Remove empty directory.

```bash
rmdir project
```

---

## find

Search files/directories.

```bash
find /home -name file.txt
```
