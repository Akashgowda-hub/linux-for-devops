# Process Management

## ps -ef

Show running processes.

```bash
ps -ef
```

---

## grep process

Search process.

```bash
ps -ef | grep nginx
```

---

## top

Real-time process monitoring.

```bash
top
```

---

## htop

Interactive process viewer.

```bash
htop
```

---

## kill

Terminate process.

```bash
kill PID
```

---

## kill -9

Force kill process.

```bash
kill -9 PID
```

---

## nproc

Show CPU core count.

```bash
nproc
```

---

## free -h

Show memory usage.

```bash
free -h
```

---

## uptime

Show system uptime.

```bash
uptime
```
