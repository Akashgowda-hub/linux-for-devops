# Archive & Compression

## Create tar.gz

```bash
tar -cvzf backup.tar.gz folder/
```

---

## Extract tar.gz

```bash
tar -xvzf backup.tar.gz
```

---

## gzip

Compress file.

```bash
gzip file.txt
```

---

## gunzip

Extract gzip file.

```bash
gunzip file.txt.gz
```

---

## zip

Create zip archive.

```bash
zip -r files.zip folder/
```

---

## unzip

Extract zip archive.

```bash
unzip files.zip
```
