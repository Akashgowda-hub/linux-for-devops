# Networking Commands

## ping

Check network connectivity.

```bash
ping google.com
```

---

## curl

Make HTTP request.

```bash
curl https://google.com
```

---

## wget

Download files.

```bash
wget https://example.com/file.zip
```

---

## ssh

Connect remote server.

```bash
ssh user@server-ip
```

---

## scp

Copy files securely.

```bash
scp file.txt user@server:/tmp
```

---

## ip a

Show IP address.

```bash
ip a
```

---

## ss -tulpn

Show listening ports.

```bash
ss -tulpn
```

---

## nslookup

DNS lookup.

```bash
nslookup google.com
```

---

## dig

Advanced DNS lookup.

```bash
dig google.com
```
