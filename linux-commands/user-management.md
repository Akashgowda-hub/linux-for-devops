# User Management

## adduser

Create new user.

```bash
sudo adduser user1
```

---

## deluser

Delete user.

```bash
sudo deluser user1
```

---

## groups

Show user groups.

```bash
groups user1
```

---

## usermod

Add user to group.

```bash
sudo usermod -aG Developer user1
```

---

## passwd

Change password.

```bash
passwd
```

---

## su

Switch user.

```bash
su user1
```

---

## sudo -i

Login as root user.

```bash
sudo -i
```

---

## id

Show UID and GID.

```bash
id
```

---

## whoami

Show current user.

```bash
whoami
```

---

## /etc/passwd

Display system user account information.

```bash
cat /etc/passwd
```
