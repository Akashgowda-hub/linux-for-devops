# Real World Linux Use Cases

Linux is widely used in DevOps and cloud environments.

---

# Web Servers

Run web applications using:

- Nginx
- Apache

Commands:

```bash
systemctl status nginx
```

---

# Docker Hosts

Linux powers container platforms.

Commands:

```bash
docker ps
```

---

# Kubernetes Nodes

Linux servers run Kubernetes workloads.

Commands:

```bash
kubectl get nodes
```

---

# CI/CD Servers

Linux is commonly used for Jenkins and GitHub runners.

---

# Log Monitoring

Monitor application logs.

```bash
tail -f /var/log/syslog
```

---

# Backup Automation

Use cron jobs for backups.

```bash
0 2 * * * backup.sh
```

---

# Cloud Infrastructure

Linux is heavily used in:

- AWS EC2
- Azure VM
- Google Compute Engine

---

# SSH Administration

Manage remote servers securely.

```bash
ssh user@server-ip
```

---

# Summary

Linux is the foundation of modern DevOps infrastructure.
